База данных для хранения объявлений о продаже автомобилей (прототип https://auto.ru/)

Список сущностей

1. Пользователь
2. Профиль
3. Каталог
4. Автомобиль
5. Объявление
6. Сообщения
7. Фотографии

В базе есть каталог автомобилей с их характеристиками.
При добавлении конкретного транспортного средства пользователем оно привязывается к записи из каталога.
При создании объявления о продаже указывается соответствующее транспортное средство, а также дополнительная информация о нём (в т.ч. фотографии).
У объявления есть несколько статусов ("Продаётся", "Продано", "Снято с продажи").
Коммуникация между продавцом и потенциальным покупателем происходит путём обмена сообщениями, к которым можно прикрепить дополнительные фотографии.
